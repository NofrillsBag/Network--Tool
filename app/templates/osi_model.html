How do you as a user, send an email? Then a recipient recieve it?

That is done by following the OSI model.

    When you want to access lets say; Youtube, what you input (send) and recieve flows down and up through the OSI model.

There are 7 layers in this model and you go up and down this model to send & recieve data, and we will go through it step by step.

  • When you are sending data this is called encapsulation
  • When you are receiving data this is called de-encapsulation
  • This is the base of how sending and recieving, data works.

    • ↳ Encapsulation (sending side data) flow, Happens first.

    Application Layer Presentation Layer Session Layer Transport Layer Network Layer Data Link Layer Physical Layer

    ↳ De-encapsulation (receiving side data) flow, Happens second.

    Physical Layer Data Link Layer Network Layer Transport Layer Session Layer Presentation Layer Application Layer
