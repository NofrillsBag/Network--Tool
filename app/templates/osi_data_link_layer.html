Data-Link Layer

Layer 2 | Data-Link Layer

Sending (Encapsulation)

This layer adds a new header containing MAC addresses, used for local delivery inside a LAN.

  • Encapsulates the Layer 3 IP packet inside a frame.
  • Adds two MAC addresses:
    • Source MAC: Your computer’s network card (NIC)
    • Destination MAC: Destination NIC (or the router, if outside network)
  • Only cares about getting data to the next hop on the local network (like a router, switch, or destination device).
  • If outside the LAN, it forwards the frame to the router’s MAC address.

Receiving (Decapsulation)

The receiving device uses this layer to verify whether the data was meant for it.

  • Checks the destination MAC address in the frame header.
  • If the MAC matches, it accepts and removes the frame header.
  • Passes the rest (the IP packet) up to Layer 3 (Network Layer).
  • If not, the frame is discarded.
