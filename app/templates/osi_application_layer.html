Application Layer

Encapsulation

  • This is the layer where apps like YouTube, Zoom, or a video game start the process of sending data.
  • It's not the app itself, it's an in between part of the app and the network.
  • It figures out what standards ( syntax) to use when talking to the network, using rules like HTTP (your browser), or VoIP (telecommunication).
  • Example: If you’re on a phone call using VoIP, this layer says “I want to send voice data” and picks a format suited for telecommunication.
  • It then passes that request down to the next layer The Presentation Layer so it can continue to be packaged and sent.

Devices

  • Web Browsers
  • Email Clients
  • Cloud Apps (Google Drive, Zoom)
  • File Transfer Tools
