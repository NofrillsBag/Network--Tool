Network Layer

Layer 3 | Network Layer (IP Address Layer)

Sending (Encapsulation)

This layer adds an IP header that lets routers deliver data across different networks it’s responsible for end-to-end delivery.

  • Encapsulates the Transport Layer segment inside an IP packet.
  • Adds two IP addresses:
    • Source IP: Your device’s IP address
    • Destination IP: The final device’s IP address
  • Routers use this info to route your packet across the world
  • These have something called a TTL (Time to Live), when packets try to find their destination, a path may not exist, TTL tells it after X, give up.

Receiving (Decapsulation)

Now the receiving device reads the IP packet to see if it’s the final destination.

  • Checks the destination IP address in the header.
  • If it matches, it removes the IP header (destination & source).
  • Passes the current information (TCP or UDP) up to Layer 4.
  • If not, and it's a router, it forwards the packet to the next hop.
