Network Layer

Encapsulation

  • The network layer adds the IP address information to your data — this is like putting a "To" and "From" address on a package.
  • This address is used to figure out the best route through the internet to reach the destination.

  • The data gets a new label called the IP header, which includes:
    • Source IP: Where the packet is coming from
    • Destination IP: Where it's supposed to go

  • The network layer works with routers — devices that decide the best path for each data packet based on these IP addresses.
  • These contain TTL (time to live), when trying to get to a destination, there may just not be a path. This tells it to give up after jumping around X times.

Examples

  • IPv4: 192.168.1.1
  • IPv6: 2001:0db8::1
  • Header Includes:
    Source IP, Destination IP, TTL (Time to Live)
  • Devices: Routers
