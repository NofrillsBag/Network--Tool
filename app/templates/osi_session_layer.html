Session Layer

Encapsulation

  • The session layer decides how long a conversation lasts between two devices. Like client and server
  • It checks if both sides are allowed to talk at the same time or if they need to take turns.
  • It also knows when to pause, resume, or end a session if needed.
  • A session is just a temporary connection — like when you log in to your online banking.
  • If you’re inactive for too long, the system might end the session and log you out. That’s the session layer doing its job.

Examples

  • Logging into a bank account
  • Video call pausing/resuming
  • Staying logged into a game
  • Live chat sessions
