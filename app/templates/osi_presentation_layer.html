Presentation Layer

Layer 6 | Presentation Layer

Sending (Encapsulation)

This layer formats the data so that a human can understand, which also depends on the type of application.

  • Converts characters, audio, images, etc. into a common structure like ASCII, MP4, or JPEG.
  • Handles compression (like zipping files or reducing video size).
  • May encrypt data for security (Like how Youtube uses HTTPS, that S means secure).
  • Makes sure data is properly structured before passing to the Session Layer.

Example: When sending a PDF, this layer converts your file into a standard format that can be viewed on any device.

Receiving (Decapsulation)

This layer translates the data back into something usable by the application.

  • Decodes formats (like turning video bytes into something a media player understands).
  • If encrypted, decrypts the message securely.
  • If compressed, expands the data back to full size.
  • Passes this clean content up to the Application Layer.

Example: When you open an image in your browser, this layer turns the raw data into a visible image.
