Presentation Layer

Encapsulation

  • This layer’s job is to make sure the data is in a format that both sides can understand.
  • If the application wants to send a photo, the Presentation layer makes sure it's in a readable format like JPEG or PNG.
  • It can also compress data (make it smaller) and encrypt it (lock it for privacy).
  • Example: You're sending a document online, this layer might compress it to a ZIP file and it may also handle encryption so only the right person can read it.
  • Once everything is in the right format, it passes the data down to the Session Layer.

Examples

  • Compressors and Decompressors (Codec) (MP3, JPEG)
  • ZIP / RAR compressors
  • SSL/TLS encryption tools
  • File format converters
